Magisk 字体模块生成器
软件说明

本软件是一个用于生成 Magisk 字体模块的工具，由于恶意 Magisk 模块可能会带来严重的安全隐患，使用本软件之前请务必确认软件来源，不要使用除 GitHub 外任何平台下载的软件。

使用帮助
  • 如果你只有一个 ttf 或 otf 文件，使用“简易单字体模式”
  • 如果你有多字重的字体家族，使用“简易多字重模式”
  • 如果你想要为英文、简体中文、繁体中文、日文、韩文等调整不同的字体，使用“高级 cjk 模式”

刷机、刷入 Magisk 模块可能会导致系统无法正常启动，请在操作前审慎考虑，并建议备份重要数据。因操作不当导致的系统故障（包括卡开机动画、功能异常、数据丢失等）或效果异常与作者无关。
